## Installation

Clone this portfolio

```bash
  git clone git@github.com:MarkkayTavion/Portfolio.git

```
    
## Deployment

To deploy this project run

```bash
  npm run deploy
```

This portfolio can be visited at https://app.netlify.com/sites/gleaming-rabanadas-c8fd16
